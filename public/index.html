About this Project

I started working on this project due to a project-work in my Allgemeinbildung-class at the BBZW-Sursee. The aim of this project was to present the company I work to my school-class. I scheduled my work and splitted my proceeding into 9 weeks of research. I wanted to show off with a presentation, out-off-the-box and started working aon an asynchronous webserver.. Additionally I searched some good page effects as the one you can see on the title page. I came up with programming a tiny project-skeleton and tryied to redesign an redevelop it. Until now the server plus this site are released and spreaded over 9 version-releases. The current one is v0.1.1pre-alpha. I started to ripp in the content at the 7th week and yeah... finally were here right? It was just a big blast... A creativity blast!

This site is hosted on my own programmed asynchronous webserver. I used coldfusion, Julia and HTML frontend. The server is serving this place at ipv4: 127.0.0.1 on port 8000. The page deployment is pretty slow and abstract. If you want to optimize the deployment-load time you should take reference to some projects such as WebKit bundling.
